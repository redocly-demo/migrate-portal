# Redocly migration helper

This repository contains a script that helps you to migrate from Redocly legacy Developer portal
to the new Product.

## Usage

1. Navigate to the root of your developer portal repository
2. Ensure that all your changes are commited and switch to some branch
3. Remove the `node_modules` directory
4. Run `npm exec github:redocly-demo/migrate-portal` and follow the instructions
